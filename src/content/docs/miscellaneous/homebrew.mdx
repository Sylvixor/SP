This section will go over how you can install homebrew apps onto your Switch.

### Requirements

---

- This section requires you to either have an **unpatched V1** Switch or a **patched V1**, **V2**, **lite** or **OLED** with **a modchip**, otherwise this will **not** work.
- This section requires you to have Atmosphere as well Hekate to be installed and for you to know how to access them.
    - If this is not the case, start by reading from <u>[FAQ](/introduction/faq/)</u>.
- This section requires you to know the different ways of accessing your microSD card.
    - If this is not the case, start by reading <u>[Browse microSD card](/important/browse/)</u>.

### Steps

---

<Steps>
    1.  ###### App

        Find the app you want to download, below you will find a list of recommended apps:

        - DBI. ( For installing titles among other tools )
        - JKSV. ( For backing up and restoring saves )
        - NX Theme Installer. ( For installing themes )
        - Fizeau. ( For adjusting things like saturation )

    2.  ###### File

        The file you should get in the end has to end in ```.nro```.

        Once you have this file of the homebrew app of your choosing, put the file into the ```/switch/``` folder that can be found on the root of your microSD card.

    3.  ###### Finish

        Now, once you boot back into AMS, you should be able to find the installed homebrew app inside of your homebrew menu.
</Steps>
